DIAGRAM: VPC Architecture
PURPOSE: Shows a VPC spanning multiple Availability Zones within an AWS Region, with subnets in each AZ demonstrating dual-stack (IPv4 + IPv6) addressing.
LAYOUT: top-to-bottom
WIDTH: 700px maximum readable width

- Availability Zone A: contains Subnet 1; border style solid, color light blue
- Availability Zone B: contains Subnet 2; border style solid, color light blue
- Availability Zone C: contains Subnet 3; border style solid, color light blue

- Subnet 3: generic subnet icon, label "Subnet: 10.0.3.0/24\nIPv6: 2001:db8:0:3::/64"

CONNECTIONS:
- No connections between subnets (they are peers within the VPC)
